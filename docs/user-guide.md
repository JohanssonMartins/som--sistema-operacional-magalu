# Manual do Usuário - S.O.M

Bem-vindo ao **Sistema Operacional Magalog**. Este guia rápido ajudará você a navegar pelas principais funções do sistema.

## 1. Acesso ao Sistema
1. Acesse o link fornecido pela gerência.
2. Na tela inicial, insira o seu **E-mail** corporativo e a **Senha**.
3. Clique em **Entrar no Sistema**.
   - *Nota:* O primeiro acesso pode demorar alguns segundos caso o servidor esteja hibernando.

## 2. Navegação Principal (Dashboard)
Ao entrar, você verá o menu lateral à esquerda com as seguintes opções principais:
- **Dashboard / Top Magalog:** Uma visão geral do desempenho dos Centros de Distribuição (CDs) baseada nas pontuações das autoavaliações preenchidas.
  - Se você for Administrador ou Auditor, poderá usar o seletor no topo do menu lateral para alternar entre "Todos os CDs" ou escolher um CD específico.
- **Autoavaliação:** A tela principal de trabalho para Gerentes Diários.
- **Base Check-List:** Uma tela administrativa para gerenciar quais itens são cobrados nas auditorias.

## 3. Realizando a Autoavaliação Diária
A aba de Autoavaliação é onde os líderes e gerentes de CD informam o status operacional da sua unidade.

1. Clique em **Autoavaliação** no menu lateral.
2. No topo da tela, você verá os **Pilares** (Ex: Pessoas, Segurança, Armazém).
3. Ao clicar na seta para expandir um pilar, você verá os **Itens de Check-List** que precisam ser respondidos.
4. Para cada item, preencha:
   - **Nota (Ponto):** Digite a pontuação atual daquele item (ex: 100, 50, 0).
   - **Nossa Ação:** Escreva um breve resumo do que a unidade está fazendo para atingir a meta ou mitigar o problema.
   - **Evidências (Opcional):** Para itens críticos, o sistema pode pedir uma foto. Clique em `Adicionar Evidência`, selecione a imagem no seu computador/celular, e aguarde o upload.
5. **Salvamento Automático:** Não se preocupe em procurar um botão "Salvar Tudo". Quando você digita uma nota ou ação, uma nuvem azul aparecerá no canto inferior direito indicando que o sistema salvou os dados automaticamente.

## 4. Gerenciamento do Perfil
Para atualizar sua foto de perfil ou sua senha:
1. Olhe para o canto inferior esquerdo do menu lateral. Você verá o seu Nome e Email.
2. **Foto:** Clique sobre a sua foto de perfil (ou sobre o ícone de usuário padrão) para selecionar uma nova foto do seu dispositivo. Ela será atualizada automaticamente.
3. **Trocar Senha:** Selecione a opção "Trocar Senha" no menu que se abre ao clicar no seu nome.
   - Digite sua Senha Atual.
   - Digite a Nova Senha (mínimo de 3 caracteres).
   - Confirme a Nova Senha e clique em **Atualizar Senha**.

## 5. Administradores: Modificando a Base
Se você tem permissão (`ADMIN` ou `AUDITOR`), você pode modificar as regras do jogo:
1. Acesse **Base Check-List**.
2. Clique em **Novo Item** para criar uma nova pergunta que aparecerá para os gerentes responderem.
3. Defina o Pilar, o Bloco, preencha a Descrição e decida se Exige Evidência.
4. Itens existentes podem ser editados (ícone de lápis) ou marcados como inativos. 
   - *Aviso:* Modificar o nome ou tipo de um item da base afetará o check-list de **TODOS** os CDs.
